# 가맹점 관리 시스템

가맹점을 위한 웹 기반 관리 시스템입니다. 가맹점주가 상품을 관리하고, 장바구니 기능 및 주문 관리를 손쉽게 수행할 수 있습니다. 관리자는 상품, 가맹점, 주문 상태 등을 중앙에서 관리할 수 있습니다.

## 주요 기능

### 가맹점주 패널
- **상품 대시보드**: 모든 상품 보기 및 카테고리별 필터링.
- **장바구니 관리**: 상품 추가, 업데이트, 삭제.
- **주문 관리**: 주문 생성 및 주문 내역 확인.

### 관리자 패널
- **상품 관리**: 상품 추가, 업데이트, 삭제.
- **가맹점 관리**: 가맹점 정보 관리.
- **주문 관리**: 가맹점에서 들어온 주문 추적 및 상태 변경.

---

## 사용 기술

- **백엔드**: Node.js, Express.js
- **프론트엔드**: HTML, CSS, JavaScript (EJS 템플릿)
- **데이터베이스**: MongoDB
- **배포**: DigitalOcean, PM2 프로세스 관리
